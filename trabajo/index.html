Manual De HTML

¿ Qué es HTML ?

HTML, sigla en inglés de HyperText Markup Language (lenguaje de marcas de hipertexto), hace referencia al lenguaje de marcado para la elaboración de páginas web. Es un estándar que sirve de referencia del software que conecta con la elaboración de páginas web en sus diferentes versiones, define una estructura básica y un código (denominado código HTML) para la definición de contenido de una página web, como texto, imágenes, videos, juegos, entre otros.

Estructura Básica de un documento HTML

Un documento HTML esta formado por etiquetas, dentro de estas escribiremos una serie de "parametros" para que el navegador las represente. Un ejemplo de una etiqueta sería este : <Etiqueta> y para cerrar la etiqueta asi :</Etiqueta>.

Todo documento HTML debe empezar con la etiqueta <!doctype html>, este indica que el documento esta escrito en HTML5.

Despues continuariamos con la etiqueta <html> que es lo que contendrá toda la informacion de la página web. A esta etiqueta se le puede añadir un atributo llamado lang donde indicaremos en que idioma esta hecha la página.
Un Ejemplo: <html lang="es"> aqui le estamos indicando que esta en español.

Luego estaria, la etiqueta <head> :
La etiqueta <head> es la cabezera del documento HTML, dentro de ella pondremos <title> aquí ira el nombre de la página, el que sale en la pestaña.
También tendra <meta charset="valor"> para indicar en que codificación esta la página, por ejemplo : <meta charset="utf-8"> para decirle que usa la codificación utf-8.
Despues añadiremos <meta name="author" content="Alejandro"> para poner el nombre del autor de la web.

La siguiente etiqueta despues de <head> sería <body>, aqui estara el cuerpo de toda la página web.

Todo esto dicho anteriormente seria la estructura basica de un documento HTML, aqui pongo un ejemplo de como quedaría: 

			<!doctype html>
			<html lang="es">
				<head>
					<title>Mi primera página</title>
					<meta charset="utf-8">
					<meta name="author" content="Alejandro">
				</head>
				<body>
					cuerpo del documento
				</body>
			</html>

Elementos de la etiqueta <body>

Dentro del cuerpo del documento tendremos las siguientes etiquetas:

<article></article>
será usada para representar un contenido específico de nuestra web. Puede representar un artículo, ya sea un artículo de opinión, un entrada en un blog, un comentario en un foro, una noticia, una descarga, o simplemente un contenido estático de una web como el típico quienes somos o a qué nos dedicamos.

<section></section>
se usa para representar un bloque de nuestra página que tiene valor semántico, es decir, que aporta un significado a la página. Esta etiqueta se coloca dentro de la etiqueta <article>.

<header></header>
Esto sera la cabezera de la página web, lo que visualizaremos en la parte de arriba de la página.

<nav></nav>
Esta etiqueta es nos creara una barra de navegación como la que hay arriba del todo de esta web, con la que podremos movernos por los diferentes sitios de la página, o nos llevara a otros sitios web.

<aside></aside>
Se usa para elementos secundarios, como podrian ser los bloques publicitarios, enlaces externos, citas, un calendario de eventos, etc. Por ejemplo si estamos hablando de un tema, con un aside podemos poner alguna informacion relacionada con ese tema, que nos ayude a entenderlo mejor.

<footer></footer>
Es el pie de página de la web.

<address></address>
Va dentro de la etiqueta <footer>, sirve por ejemplo para colocar una dirección de email para contacto.

Un ejemplo:

<p> Esto es otro parrafo </p>

Listas

Los elementos de una lista, se hacen con la etiqueta <li>, para formar una lista desordenada, utilizaremos <ul> y para una ordenada usaremos <ol>
Ejemplo: 

			<ol>
				<li> una </li>
				<li> lista </li>
				<li> ordenada </li>
			</ol>
			<ul>
				<li> una </li>
				<li> lista </li>
				<li> desordenada </li>
			</ul>
  1. una
  2. lista
  3. ordenada
  • una
  • lista
  • desordenada

Luego tenemos las listas de definición, que es parecido a lo que usariamos por ejemplo en un diccionario. Para crear la lista se usa la etiqueta <dl> y dentro irian <dt> y <dd>, pongo un ejemplo para que se entienda su manejo: 

			<dl>
				<dt>
					Mariposa
				</dt>
				<dd>
					Insecto de boca chupadora, con dos pares de alas cubiertas de escamas y generalmente 
					de colores brillantes, que constituye la fase adulta de los lepidópteros.
				</dd>
				<dt>
					León
				</dt>
				<dd>
					Gran mamífero carnívoro félido africano, de pelaje amarillo rojizo, con la cabeza 
					grande, los dientes y las uñas muy fuertes, la cola larga y terminada en un fleco 
					de cerdas, y cuyo macho se distingue por una larga melena.
				</dd>
			</dl>
Mariposa
Insecto de boca chupadora, con dos pares de alas cubiertas de escamas y generalmente de colores brillantes, que constituye la fase adulta de los lepidópteros.
León
Gran mamífero carnívoro félido africano, de pelaje amarillo rojizo, con la cabeza grande, los dientes y las uñas muy fuertes, la cola larga y terminada en un fleco de cerdas, y cuyo macho se distingue por una larga melena.

En la etiqueta de las listas ordenadas <ol> podremos ponerle diferentes atributos, por ejemplo para que el orden de la lista salga inverso, o que salga ordenado por letras, o por numeros romanos, para ello hay que poner <ol reversed> para que salga en orden invertido, y <ol type="tipo"> para ponerlo en numeros romanos, letras, dependiendo de que tipo le pongamos.

  • <ol type= -1>: nos creara una lista ordenada con numeros, es la que viene por defecto.
  • <ol type= -a>: nos creara una lista ordeada con letras minusculas.
  • <ol type= -A>: nos creara una lista ordenada con letras mayusculas.
  • <ol type= -i>: nos creara una lista ordenada con numeros romanos en minuscula.
  • <ol type= -I>: nos creara una lista ordenada con numeros romanos en mayuscula.

Tablas

Para crear tablas utilizaremos las siguientes etiquetas:

<table>, </table> dentro de esta etiqueta es donde escribiremos el cuerpo de nuestra tabla
con la etiqueta <tr> crearemos las columnas
y con la etiqueta <td> las filas
si a la etiqueta table le añadimos el atributo border añadiremos un borde a la tabla, ya que por defecto sale sin borda.

Un ejemplo de una tabla:

				</tr>

					<td>Halo

					<td>Xbox
				</tr>
				<tr>
					<td>Final Fantas 8
					<td>Rpg
					<td>Play Station
				</tr>
			</table>

Un ejemplo:

Elementos Semánticos de Texto

A los textos que escribamos podremos resaltarlos de diferentes formas con las siguientes etiquetas:

  • <i> : pondremos lo que este dentro en cursiva.
  • <b> : pondremos lo que este dentro en negrita.
  • <u> : pondremos lo que este dentro subrayado.
  • <EM> : resaltaremos una parte del texto.
  • <STRONG> : Pondremos como en negrita el texto.
  • <SMALL> : Pondremos la letra pequeña.
  • <S> : Pondremos el texto tachado.

Ahora pondremos algunos ejemplos:

<i>Esta frase esta en cursiva</i>. <b>Esta en negrita</b>. <u>Esta subrayada</u>. <EM> Esta resaltada </EM>. <STRONG>Esta en STRONG</STRONG>. <SMALL>Esta en letra pequeña</SMALL>. <S>y esta tachada</S>

Esta frase esta en cursiva. Esta en negrita. Esta subrayada. Esta resaltada . Esta en STRONG. Esta en letra pequeña. y esta tachada

Hiperenlaces

Con la etiqueta <a href="url de la página web"> podremos poner enlaces a otras páginas webs,o ficheros html que esten en nuestro directorio o subdirectorios si estamos trabajando de forma local, un ejemplo:

<a href="https://www.google.es/">Aquí para ir a google</a>

Aquí para ir a google

Tambien podemos usarlo para movernos por la propia pagina web, colocamos la siguiente etiqueta <a name="nombre"> con eso haremos referencia a otra parte de la página, luego en cualquier etiqueta de las creadas de toda la página le añadimos por ejemplo <p id="nombre">, entonces al pinchar al texto que le hayamos asignado la etiqueta <a name="nombre"> nos llevará a donde hallamos colocado la id, en este caso donde pusimos <p id="nombre">
Si ponemos lo siguiente <a href=#top> ir al principio de la página </a> nos llevara al principio de la página

Contenido encajado

Con la etiqueta <img src="ruta imagen" alt="mi foto"> podemos añadir imagenes a nuestra web, mediante una url de internet, o desde una imagen que tengamos en alguna carpeta en local. Un ejemplo

<img src="https://halo.bungie.net/images/dark/projects/games/halo/games_halo_header.jpg" alt="Foto de halo" >

Mapas de imagen

Un mapa de imagen es, al tener una imagen, y pinchar en algun lado en conreto, que te lleve por ejemplo, a una pagina web. Por ejemplo, tienes una imagen donde tienes caratulas de videojuegos, cuando pinches en una de las cartulas, te llevara a su respectiva página oficial.

Un ejemplo de como se hace:

<img src="ruta de la imagen" usemap="id">
<map name="nombre">
<area shape=rect coords= 45,45,45,45 href="url de la pagina" alt="nombre">
</map>

Dependiendo de que forma le quieras dar al area, pondremos diferentes atributos, por ejemplo si el area que queremos hacer es un rectangulo, pondremos rect, si es un circulo, circle, si es un poligono, poly.
En coords añadiremos las diferentes coordenadas para situar el area donde nosotros queramos.
