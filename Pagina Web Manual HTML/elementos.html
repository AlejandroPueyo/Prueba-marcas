Elementos

	Los elementos consisten en abrir y cerrar etiquetas con contenido dentro.
	
	Por ejemplo:

	<p>Mi primer parrafo</p>

	Tambien los puede haber sin contenido, que se llaman elementos vacios.
	
	<p></p>

	Los elementos pueden estar dentro de otros elementos, estos se llaman elementos anidados.
	
	Por ejemplo, el body esta dentro del elemento html.

	

	
Los elementos mas comunes son:

	
Elemento <body>

	
Este elemento define el cuerpo del elemento.

	
	
Elemento <h1>

	
Este elemento define un encabezamiento.

	
Hay varios tipos de encabezamiento, desde el numero 1 al numero 6.


	
Elemento <p>

	
Este elemento define un parrafo.


	
Elemento <pre>

	
Este elemento define un parrafo pero que respeta las tabulaciones, los saltos de linea, etc...


	
Elemento <a>

	
Este elemento define un enlace.

	
	
Elemento <br>

	
Este elemento define un encabezamiento.

Elemento <dt>

Elemento <dd>

Elemento <head>

Elemento <img>

Elemento <meta>

	
Este elemento define un metadato sobre el documento.


	
Elemento <nav>

	
Este elemento define menu para navegar entre diferentes enlaces.

	
	
Elemento <ol>

	
Este elemento define una lista ordenada.


	
Elemento <table>

	
Este elemento define una tabla.


	
Elemento <title>

	
Este elemento define un titulo para el documento.


	
Elemento <ul>
