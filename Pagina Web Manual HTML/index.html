Manual HTML

¿Que es HTML?

HTML es el lenguaje de marcado estandar para la creacion de paginas Web.
  • El codigo HTML describe la estructura de las paginas web usando el marcado.
  • Los elementos de HTML son los bloques de construccion de las paginas HTML.
  • Los navegadores no muestran las etiquetas HTML, pero las utilizan para representar el contenido de la pagina.
