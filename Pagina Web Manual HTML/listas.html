Listas

	Hay dos tipos de listas:
	
    
		
  • Listas ordenadas
    • 
		
  • Listas desordenadas
    • 
	
	

	Las listas ordenadas las creamos con la etiqueta <ol> y las listas desordenadas con <ul>.
	Dentro de ellas debemos utilizar <li> para enumerar las cosas que queremos poner.
	En las listas desordenadas podemos especificar el estilo con el atributo style y el estilo que queremos darle.
	Los estilos disponibles son: disc, circle, square y none.
	
	
	

	En las listas ordenadas, tenemos los estilos: 1, A, a, I e i.
