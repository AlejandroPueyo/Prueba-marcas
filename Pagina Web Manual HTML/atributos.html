Atributos de los elementos

	Los atributos de los elementos son diferentes opciones que tienen los elementos.
	Se deben especificar el la etiqueta de inicio y tiene la estructura nombre="valor".
	
	Algunos de ellos son:

	
alt

	
Este atributo funciona para los elementos <area>, <img>, <input>.

	
Este especifica un texto alternativo cuando el elemento original falla.

	
	
charset

	
Este atributo funciona para los elementos <meta>, <script>.
	
	
Este especifica la codificacion de caracteres.
	
		
	
title

	
Este atributo funciona para el elemento <p>.

	
Añade un titulo al parrafo.


	
href

	
Este atributo funciona para el elemento <a>.

	
Especifica la direccion del enlace.


	
src

	
Este atributo funciona para el elemento <img>.

	
Especifica la ruta de la imagen.
