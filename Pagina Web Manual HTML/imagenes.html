Imagenes

	Para insertar imagenes en un documento HTML utilizamos el elemento <img>.
	Este elemento solo contiene atributos, no contiene contenido en el.
	Ademas este elemento no se cierra.
		
	

	En los atributos tambien podemos especificar las dimensiones que queramos que tenga.
	Se puede indicar el pixeles, cm, mm, en porcentaje, etc...

	

	A una imagen le podemos marcar diferentes zonas de ella que cada zona tenga un enlace.
	Para hacer esto usamos el atributo usemap dentro del elemento <img>.
	Despues utilizamos el elemento <map> con el atributo name y que tenga como valor el nombre que le hayamos dado.
	Dentro de este elemento, abrimos otro elemento <area>, que no hace falta cerrarla, y en este elemento ponemos los 
	atributos shape para indicar la forma de la zona (circle, rect, etc..), las cordenadas de la zona con coords y el enlace con href.
