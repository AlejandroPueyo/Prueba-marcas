Tablas

	Para hacer una tabla usamos el elemento <table>.
	Dentro de el, usamos <tr> para definir una fila, 
	<td> para una columna y <th> para definir un encabezado de la tabla.

	
	
	En una tabla podemos unir columnas utilizamos el atributo colspan y el numero de celdas que quieres unir.

	
	
	Y para unir filas utilizamos rowspan y el numero de filas que queremos unir.

	

	Si queremos darle estilos lo haremos mediante el CSS.
